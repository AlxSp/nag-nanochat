# Figure scripts

Every figure in the [blog post](https://alexanderspeicher.com/blog/norm-agnostic-residual-networks)
is generated by the scripts in this directory. Outputs land in `dev/figures/output/` (gitignored).

## Prerequisites

Most scripts probe the trained checkpoints. Download them from
[HF: AlxSp/nag-nanochat-d64](https://huggingface.co/AlxSp/nag-nanochat-d64) into `<repo>/checkpoints/`:

```bash
hf download AlxSp/nag-nanochat-d64 --local-dir checkpoints/
```

Scripts that build an evaluation batch also need the tokenizer and a ClimbMix validation shard
under `NANOCHAT_BASE_DIR` (defaults to `~/.cache/nanochat`). Everything runs on CPU in minutes;
no GPU required.

## Script → figure map

| script | figures |
|---|---|
| `generate_campaign_plots.py` | headline val-bpb curves (R1), gate-health quartiles (R2), gap-vs-budget (R3), ablation ladder (R4), bpb/CORE split (R5), diagram panels (P1–P3) |
| `generate_training_loss_plots.py` | training-loss curves incl. the collapsed run (needs W&B history / local run logs) |
| `generate_fig3_residual_geometry.py` | residual norm & cumulative rotation vs depth, GPT vs NAG |
| `generate_fig3_norm_robustness.py` | BOS-outlier decomposition of the norm curve (mean vs ex-BOS vs median) |
| `generate_fig3_residual_geometry_median.py` | median-based companion of the norm/rotation figure |
| `generate_nag_mechanism_figures.py` | layer-wise α and realized gain (paper Fig 5), preferred-direction similarity (Fig 6), norm-as-confidence calibration |
| `generate_norm_confidence_with_gpt.py` | confidence calibration with GPT entropy reference |
| `generate_fig7_attention_sink.py` | attention-sink probe (BOS attention mass, GPT vs NAG) |
| `generate_nag_token_norm_plots.py` | per-token norm trajectories over example sequences (P4) |

Run any of them from the repo root:

```bash
uv run python dev/figures/generate_nag_mechanism_figures.py
```
